Welcome to Drei Chinesen mit dem Kontrabass

This site is based on a children's game where you replace the vowels in a rhyme/song by one vowel and try to sing the rhyme. This site allows you to replace any character (not only vowels) by one or more other characters.

Enter the text to transform into the following field


Choose the letters to replace (Separate multiple letters by comma, only single characters)


Choose one or more letters to replace the above letters with. If you choose more than one letter, you will get multiple answers. You can also specify something like 'blub'





Check this option to remove sequences of the same letter in the generated text.

Check this option if you use a multi-letter replacement, unless you want repetitions of the combination
